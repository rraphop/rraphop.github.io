# 사회·역사 배움터 프로젝트 요약

## 프로젝트 개요

현재 프로젝트는 중학교 사회·역사 교과 학습 홈페이지입니다. 2015·2022 개정 교육과정 흐름을 바탕으로 홈페이지 소개, 단답형 퀴즈, 학습 게임, 질문과 답변, 교육과정 안내를 제공하는 정적 웹사이트입니다.

## 구성 파일

- `index.html`: 메인 화면, 히어로 영역, 방문자 카운터, 주요 기능 및 게임 바로가기
- `intro.html`: ABOUT 페이지, 홈페이지 소개와 주요 학습 흐름 안내
- `quiz.html`: 사회·역사 단답형 퀴즈 화면
- `game.html`: 역사 연표 배열 게임과 산성비 게임 화면
- `qna.html`: 질문 등록, 질문 목록, 상세 보기, 수정, 관리자 답변/삭제 화면
- `curriculum.html`: 2015·2022 교육과정 안내와 비교표
- `style.css`: 전체 레이아웃, 반응형 스타일, 퀴즈/게임/Q&A UI 스타일
- `script.js`: 메뉴, 방문자 카운터, 퀴즈, 게임, Q&A 동작 로직
- `quiz-data.js`: 사회1(2022), 사회2(2015), 역사1(2022), 역사2(2015) 퀴즈 데이터
- `assets/social-history-hero.png`: 메인 히어로 이미지

## 현재 구현 기능

### 공통 UI

- 상단 로고는 `中 사회·역사 배움터`입니다.
- 상단 메뉴는 `ABOUT`, `퀴즈`, `게임`, `질문과 답변`, `교육과정 안내`입니다.
- 현재 페이지 메뉴에 활성 표시가 적용됩니다.
- 모바일 화면에서는 메뉴 버튼으로 내비게이션을 열고 닫습니다.

### 메인 페이지

- 중학교 사회·역사 교과 학습 홈페이지임을 첫 화면에서 안내합니다.
- `퀴즈 풀기`, `게임 선택`, `교육과정 보기` 바로가기가 있습니다.
- 오늘 방문자 수와 총 방문자 수를 표시합니다.
- 방문자 수는 Apps Script를 거쳐 Google Sheets `count` 시트에 날짜별 1행으로 저장됩니다.
- 오늘 방문자 수는 Apps Script의 `Asia/Seoul` 기준 하루(00:00:00~23:59:59) 동안 누적됩니다.
- `count` 시트의 `startTotal` 값으로 오늘 방문자 수를 `total - startTotal` 기준으로 복구할 수 있습니다.
- 방문자 카운터는 마지막으로 받은 숫자를 브라우저에 임시 캐시해 먼저 표시한 뒤, Google Sheets 응답 값으로 다시 갱신합니다.
- 오늘 방문자 수와 총 방문자 수가 길어져도 영역 안에 맞도록 글자 크기를 조정합니다.
- 사회 산성비, 역사 산성비, 역사 연표 배열 게임 바로가기가 있습니다.

### ABOUT 페이지

- 상단 메뉴명은 `ABOUT`로 유지합니다.
- 본문 제목은 `홈페이지 소개`로 유지합니다.
- 홈페이지 소개, 학습 목표, 추천 사용법을 안내합니다.
- `퀴즈풀기`, `게임하기`, `질문하기` 바로가기 3칸이 있습니다.

### 퀴즈

- 과목 구분:
  - 사회1(2022개정)
  - 사회2(2015개정)
  - 역사1(2022개정)
  - 역사2(2015개정)
- 단원을 선택한 뒤 단답형 5문항이 무작위로 출제됩니다.
- 같은 단원에서 최대 30문항까지 이어서 풀이할 수 있습니다.
- 객관식은 없고 단어, 용어, 빈칸형 중심입니다.
- 한 용어에서 여러 문제 유형을 생성합니다.
- 공백, 일부 문장부호, 대소문자 차이는 정답 비교에서 무시합니다.
- 채점 후 입력 답안, 정답, 해설을 보여줍니다.

### 게임

- 게임 선택 화면에서 `역사 연표 배열`과 `산성비 게임`을 전환할 수 있습니다.
- URL 해시로 산성비 게임 바로가기도 처리합니다.

#### 역사 연표 배열 게임

- `한국사`와 `세계사` 모드를 선택할 수 있습니다.
- 사건 은행에서 10개 사건을 무작위로 뽑습니다.
- 사건 카드를 오래된 사건에서 최근 사건 순서로 배열합니다.
- 위/아래 이동 버튼으로 순서를 조정합니다.
- 순서가 모두 맞으면 합격 메시지를 보여줍니다.
- 틀리면 정답 순서를 보여줍니다.
- 교과서 역사 사건 은행을 펼쳐 볼 수 있습니다.

#### 산성비 게임

- 산성비 게임 안에서 `사회`와 `역사` 항목을 선택할 수 있습니다.
- 스페이스바 또는 시작 버튼으로 게임을 시작합니다.
- 떨어지는 사회·역사 용어를 입력하고 Enter를 누르면 제거됩니다.
- 점수, 생명, 단계, 생존시간을 표시합니다.
- 점수가 오르면 단계가 상승하고 용어가 더 빠르게 떨어집니다.
- 생명이 0이 되면 게임이 종료됩니다.
- 게임 종료 후 이름을 입력해 랭킹을 등록할 수 있습니다.
- 사회 산성비 랭킹과 역사 산성비 랭킹을 Google Sheets의 별도 시트에 따로 저장하며 각 상위 10개 기록을 표시합니다.
- 산성비 용어 은행은 `quiz-data.js`의 사회·역사 용어를 우선 사용하고, 데이터가 없으면 기본 용어 목록을 사용합니다.

### 질문과 답변

- 질문 등록 시 `소속`, `학년`, `이름`, `질문 내용`, `수정 비밀번호`를 입력합니다.
- 소속, 학년, 이름은 각각 익명 체크가 가능합니다.
- 질문 내용은 비공개 체크가 가능합니다.
- 등록된 질문은 Apps Script를 거쳐 Google Sheets `QNA` 시트에 저장됩니다.
- 질문 목록은 10개씩 페이지네이션됩니다.
- 질문을 선택하면 상세 보기 화면에서 질문과 답변을 확인합니다.
- 등록자가 설정한 수정 비밀번호로 질문 내용을 수정할 수 있습니다.
- 관리자 비밀번호로만 답변을 저장할 수 있습니다.
- 관리자 비밀번호로만 질문을 삭제할 수 있습니다.
- 관리자 비밀번호는 `teacher1234`입니다.

## 저장 방식

- 이 프로젝트는 정적 웹사이트이며, 질문 게시판, 방문자 수, 산성비 랭킹은 Google Sheets + Apps Script 웹앱을 통해 공유 저장됩니다.
- 방문자 수는 `count` 시트에 날짜별 1행으로 저장되며, 같은 날짜 방문은 해당 행의 오늘 방문자 수로 누적됩니다.
- 산성비 랭킹은 `사회 산성비 랭킹`, `역사 산성비 랭킹` 시트에 저장되며, 브라우저 저장소는 마지막으로 받은 랭킹을 임시 캐시하는 용도로만 사용됩니다.

## 최근 주의한 점

- 상단 메뉴의 홈페이지 소개는 `ABOUT`로 유지합니다.
- 소개 페이지 본문 제목은 `홈페이지 소개`로 유지합니다.
- 기존 `史` 로고는 중학교 의미의 `中`으로 변경되어 있습니다.
- `style.css`, `script.js`, `quiz-data.js` 파일명은 정확히 유지해야 합니다.
- 관리자 비밀번호 `teacher1234`는 현재 프론트엔드 코드에 직접 들어 있습니다.
- 서버 저장이 필요한 실제 운영용 게시판으로 확장하려면 Q&A와 랭킹 저장 구조를 별도 백엔드로 옮겨야 합니다.
